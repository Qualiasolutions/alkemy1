Production Workflow

01. Initialization

A. Script Creation

The workflow begins with the script, established via one of three methods:

  • Import an existing file.
  • AI-generate a script.
  • Manually write the script within the app.

B. Moodboard Generation

Curate a visual style by importing images or using the embedded web search.

  • Outcome: After selecting/importing 10 images or videos, the system auto-generates the project aesthetic.

02. Asset Proposal

Based on the Script and Moodboard, the system automatically generates:

Auto-Generated Assets

  • Actors: 10 different proposed personas (including casting videos) derived from the script characters.
  • Locations: 10 different environment options for each scene to serve as the base for the 3D world.

Customization

The user can select a proposed asset, define a new persona, or manually edit the proposed items.

03. LoRA File Creation

Finalized choices are trained into LoRA files to ensure consistency.

Character LoRA

  • User triggers auto-generation.
  • System captures the persona in various angles, lighting, and aspects.
  • Resulting file is automatically imported to Composite/Presentation.

Location LoRA (3D World)

  • User selects a generated location to enter the 3D world.
  • Capture Process:
    • Auto: System generates dataset images.
    • Manual: User navigates using Unreal Engine tools (angles/lenses) to take snapshots.
  • User reviews and finalizes the dataset.
  • Resulting file is automatically imported to Composite/Presentation.

04. Composition

The Composite tab serves as the staging ground.

  • Auto-Setup: System reads the script and places suitable LoRA Personas and Locations together.
  • Prompting: User prompts the model to produce keyframes.
  • Keyframe Requirements:
    • First Frame (Required).
    • End Frame (Optional).
  • Action: Upscale the chosen frame to proceed to Animation.

05. Animation

  • Input: Upscaled shot from the Composite step.
  • Process: User selects a model and writes a prompt to animate the static shot.
  • Completion: Once satisfied, the user upscales the generated video to prepare for the timeline.

06. Timeline & Assembly

Import Logic

User can send a single specific video or all generated variations to the timeline.

Timeline System

The timeline automatically organizes clips in sequential order based on data from the Composite tab and Script.

Editing Feature

Regenerate Shot: A specific clip can be regenerated, automatically replacing the old shot on the timeline.
