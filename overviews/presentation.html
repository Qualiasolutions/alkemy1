Presentation System

Automatic Pitch Deck Generation from Production Assets

What is the Presentation System?

The Presentation System automatically generates professional pitch decks from your production assets. Every element in your project - from moodboards to storyboards - updates the presentation in real-time, ensuring stakeholders always see the latest creative vision.

How It Works

1
Asset Collection
System automatically gathers assets from all tabs: script analysis, moodboards, character/location visuals, and scene storyboards.
2
Professional Layout
Content arranged in production-standard format: Title & Logline, Visual References, Cast, Locations, Scene-by-Scene Storyboard.
3
Interactive Gallery
Click any character or location to enter fullscreen image gallery. Navigate with arrow keys, view all generated variants.
4
Real-Time Updates
Presentation automatically reflects changes. Add new moodboard images, refine characters, or update scenes - deck stays synchronized.

Presentation Sections

Title & Logline

Project name and AI-generated one-sentence pitch extracted from script analysis.

Moodboard

Visual references organized by category with AI summaries. Auto-carousel cycles through references every 4 seconds.

Cast

Character cards with hero images, descriptions, and full image galleries. Hover for gallery access prompt.

Locations

Filming location visuals with setting details. Click to view full resolution and explore all variants.

Storyboard

Scene-by-scene shot breakdown with visual previews. Shows shot number, description, and hero frame image.

Interactive Features

Fullscreen Image Gallery

  • Click character or location card to enter gallery mode
  • Arrow keys navigate through all generated variants
  • ESC key returns to main presentation view
  • Image counter shows current position (e.g., "3 / 12")
  • Includes hero images, generations, and refined outputs

Moodboard Carousel

  • Auto-rotates through reference images every 4 seconds
  • Smooth cross-fade transitions between references
  • Displays AI-generated visual language summaries
  • Supports both legacy moodboard and new template system
  • Shows reference count per category
