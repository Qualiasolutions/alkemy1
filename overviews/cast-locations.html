Cast & Locations System

AI-Powered Visual Development with Character Identity Training

What is Cast & Locations?

The Cast & Locations system is your visual development studio for characters and filming locations. Generate photorealistic images with multiple AI models, train custom character identities for 90-98% visual consistency, and build a complete visual library for your production.

Character Generation Workflow

1
Select Character
Choose from script-analyzed characters or create custom entries. Click card to open Generation Studio.
2
Configure Generation
Select AI model (Flux Pro, Gemini, Imagen), aspect ratio, number of variants (1-10), and attach reference images.
3
Generate Variants
AI creates multiple visual options based on character description and moodboard context. 15-45 seconds per variant.
4
Review & Select
Compare variants side-by-side in grid view. Select hero image or enter Refine Studio for improvements.
5
Train Identity (Optional)
Upload 6-12 reference images to train custom LoRA model. Ensures character consistency across all future generations.

Character Identity Training

LoRA (Low-Rank Adaptation) technology enables training custom character models that maintain 90-98% visual similarity across all generations.

1
Prepare References
Collect 6-12 high-quality images of character from multiple angles and lighting conditions.
2
Upload & Train
Drag images into Character Identity Modal. Fal.ai trains custom Flux LoRA model (5-10 minutes).
3
Visual Testing
System generates test images to validate consistency. AI rates similarity score (target: 90-98%).
4
Auto-Integration
Trained LoRA automatically applied to all future generations featuring this character. Zero manual setup required.

AI Models Available

Flux Pro 1.1

Highest quality photorealistic images via Fal.ai. Best for final hero images and client presentations.

Flux Dev

Fast iterations with character identity support. Best for rapid exploration and refinement cycles.

Gemini Flash Image

Multimodal editing with reference images. Best when working with existing visuals and refinements.

Imagen 4.0

Google's text-to-image model. Best for pure text-based generation without references.
