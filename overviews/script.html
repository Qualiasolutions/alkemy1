Script Analysis System

AI-Powered Screenplay Breakdown for Film Production

What is Script Analysis?

The Script Analysis system is your production entry point. Upload any screenplay and AI automatically extracts all essential production elements - scenes, characters, locations, props, and technical requirements - transforming hours of manual breakdown work into seconds.

How It Works

1
Upload Screenplay
Drag and drop or paste screenplay in PDF, TXT, Markdown, or Fountain format. Advanced PDF parsing extracts text with structure preservation.
2
Text Extraction
System parses file format and extracts clean text content. Real-time progress feedback during processing.
3
AI Analysis
Google Gemini 2.5 Pro analyzes screenplay structure, identifying scenes, characters, dialogue, and action lines using natural language understanding.
4
Structured Output
Results displayed in organized sections with production-ready categorization. Auto-generated logline and professional synopsis included.

What Gets Extracted

Scenes

Setting, time of day, mood, lighting specs, summary, and camera package requirements for each scene

Characters

Names, descriptions, character arcs, and dialogue patterns automatically identified

Locations

Setting details, visual requirements, and environmental characteristics for each filming location

Props

Physical objects mentioned in script that need acquisition or creation for production

Styling & Wardrobe

Costume requirements and character appearance specifications per character and scene

Set Dressing

Environmental details, decorations, and atmosphere elements for each location

Makeup & Hair

Character appearance details, aging effects, wounds, and visual transformation requirements

Sound Cues

Audio requirements, music specifications, and sound design notes extracted from action lines

AI Processing

Technology Stack

Google Gemini 2.5 Pro provides deep screenplay understanding with natural language processing that recognizes screenplay conventions like INT/EXT scene headings, character dialogue formatting, and action line structure.

Fallback System

Automatic model fallback chain ensures reliability: Gemini 2.5 Pro → Gemini 2.5 Flash → Gemini 2.0 Pro. If all APIs are unavailable, system provides demo mode with sample data.

Smart Parsing

  • Recognizes standard screenplay format conventions
  • Identifies scene headings (INT/EXT, location, time)
  • Extracts character names from dialogue blocks
  • Parses action lines for props and technical requirements
  • Generates structured JSON output for downstream processing

Supported Formats

PDF Documents

Advanced OCR-like parsing using pdf.js library. Handles formatted screenplays from Final Draft, Celtx, and other industry tools.

Plain Text (.txt)

Direct text file upload with instant parsing. Best for quick testing and simple screenplay formats.

Markdown (.md)

Markdown-formatted screenplays with header recognition and structure preservation during parsing.

Fountain Format

Industry-standard plain text screenplay format. Full support for Fountain syntax conventions.
