Key Services

  • aiService.ts: Centralized AI API interactions (script analysis, image/video generation, Director assistant)
  • analyticsService.ts: Quality analysis and performance metrics tracking (627 lines)
  • voiceService.ts: Web Speech API integration for voice input/output (297 lines)
  • worldLabsService.ts: Enterprise-grade 3D world generation with Gaussian Splatting
  • videoRenderingService.ts: Client-side video rendering using FFmpeg.wasm
  • imageSearchService.ts: Multi-source AI-enhanced image search (Pexels, Unsplash, Brave)
  • directorKnowledge.ts: Cinematography knowledge base (DOF calculations, lens recommendations)
