# Story 7a.4: Community Tutorials and Highlights

**Epic**: Epic 7a - Community Hub (Growth)
**PRD Reference**: Section 6, Epic 7a, Story 7a.4
**Status**: Not Started
**Priority**: Low (V2.2 Growth Feature)
**Estimated Effort**: 4 story points
**Dependencies**: Story 7a.1 (Film Gallery)
**Last Updated**: 2025-11-09

---

## User Story

As a **filmmaker**,
I want to **access tutorials and see highlighted community work**,
So that **I can learn techniques and stay inspired**.

---

## Business Value

Tutorials reduce support burden, improve user success rates, and showcase platform capabilities. Highlights celebrate community excellence and drive engagement.

**Success Metric**: >40% of new users view at least 1 tutorial; tutorials reduce support tickets by 20%.

---

## Key Acceptance Criteria

### AC1: Tutorial Library
- Curated tutorial collection (video + text guides)
- Categories: Getting Started, Character Design, 3D Worlds, Audio Production, Advanced Techniques
- Search and filter functionality
- Tutorial ratings and feedback

### AC2: Community News
- Announcements section (new features, competitions, updates)
- Chronological feed with timestamps
- Admin-only post creation

### AC3: Featured Work Section
- Admin-curated "Film of the Week"
- "Trending Now" algorithmic section (most views/likes in last 7 days)
- "Staff Picks" for exceptional work

### AC4: Tutorial Contributions
- Community members can submit tutorial proposals
- Admin review and approval workflow
- Tutorial creator attribution and profile link
